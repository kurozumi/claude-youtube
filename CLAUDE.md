# YouTube チャンネル成長戦略

## 基本設定

| 項目 | 内容 |
|------|------|
| チャンネル名 | じゅんこのリアル副業日記 |
| 運営者 | じゅんこ（真面目・純粋な性格） |
| ニッチ | ビジネス・副業 |
| ターゲット | 主婦・シニア層 |
| 目標 | 3ヶ月で登録者1,000人達成 |

## チャンネルポジション

> **「50代・主婦でもできる、スマホ1台で月3万円稼ぐ副業チャンネル」**

- 「副業」だけでは競合が多い → 年齢・属性で絞り込み差別化
- 主婦・シニアの「私でもできるの？」という不安に共感ファーストで訴求
- 「現実的・低リスク・スマホだけ」がターゲットに刺さる

---

## コンテンツの3本柱

### 柱1：入門・基礎（全体の40%）
検索流入を最大化する"教科書"コンテンツ

- フリマアプリで月1万円稼ぐ完全手順【主婦向け】
- スマホだけでできる副業5選【50代でも安心】
- メルカリ初心者が最初にやること10ステップ
- 副業詐欺の見分け方【シニアが狙われやすい手口】

### 柱2：実践・体験（全体の35%）
視聴者の「私もできそう」を引き出す行動喚起コンテンツ

- 【実演】家にある不用品を売ってみた結果
- 1週間副業に挑戦した記録（収益公開）
- ハンドメイド販売で初めて売れた瞬間
- 視聴者さんが稼いだ方法を聞いてみた

### 柱3：Q&A・悩み解決（全体の25%）
コミュニティ形成と継続視聴を促すエンゲージメントコンテンツ

- 「子育て中でも副業できますか？」に答えます
- コメント欄から質問に回答！副業相談室
- 副業を始めて失敗したこと正直に話します
- 主婦が副業で稼いだお金の使い道【家族の反応も】

## 3ヶ月ロードマップ

### フェーズ1：基盤構築（1〜4週目）
**週3本投稿（月・水・金）**

| 週 | テーマ | 目的 |
|----|--------|------|
| 1週目 | チャンネル紹介 + 副業入門① | ポジション表明 |
| 2週目 | 検索ボリュームが高い基礎動画 × 3本 | SEO流入獲得 |
| 3週目 | 実践・体験系 × 3本 | 視聴維持率向上 |
| 4週目 | Q&A + まとめ系 × 3本 | コメント促進 |

**KPI目標:** 動画12本・登録者50〜100人

### フェーズ2：アルゴリズム攻略（5〜8週目）
**週2〜3本（質を優先）**

- 再生数が伸びたジャンルを特定して深掘り
- サムネイル・タイトルをA/Bテスト感覚で検証
- 視聴者維持率60%超えを動画ごとに確認

**KPI目標:** 登録者200〜400人

### フェーズ3：加速（9〜12週目）
**週2〜3本 + Shorts 週3本**

- Shortsで短尺流入を追加（既存動画のダイジェスト切り出しでOK）
- コメント返信を徹底してコミュニティ感を醸成
- 「登録者限定プレゼント」など登録促進施策を実施

**KPI目標:** 登録者700〜1,000人

## 数値目標

| 時期 | 動画本数 | 登録者 | 総再生時間 |
|------|---------|--------|-----------|
| 1ヶ月後 | 12本 | 50〜150人 | 200時間 |
| 2ヶ月後 | 24本 | 300〜500人 | 800時間 |
| 3ヶ月後 | 36本+ | **1,000人** | 4,000時間 |

> 収益化条件（登録者1,000人 + 総再生時間4,000時間）を3ヶ月で同時達成する設計

## 勝ちパターン

### タイトル・サムネイルの鉄則

```
【良い例】「50代主婦がメルカリで月3万稼いだ方法」
【悪い例】「副業について解説します」

ポイント：
・数字を必ず入れる（月3万・5選・10ステップ）
・ターゲット属性を明示（50代・主婦・シニア）
・サムネイルは大きい文字3行以内
```

### 最初の30秒の構成

```
0〜5秒  ：「〇〇に悩んでいませんか？」（共感フック）
5〜15秒 ：「この動画を見ると〇〇がわかります」（約束）
15〜30秒：「私は〇〇な主婦です」（信頼性確立）
```

### 狙うべきキーワード（SEO）

```
・副業 主婦 スマホ
・副業 50代 初心者
・メルカリ やり方 初めて
・在宅ワーク シニア おすすめ
→ 競合が少なく需要がある「ロングテール」狙い
```

### コミュニティ戦略

- 毎動画コメントに全返信（最初の100本は必須）
- 「コメントで質問してね」を動画内で呼びかける
- LINEオープンチャットやX(Twitter)と連携

## チャンネル開設チェックリスト

- [ ] チャンネル名にターゲット属性を入れる（例：「主婦のあきこ副業チャンネル」）
- [ ] チャンネルアート・アイコンに顔写真を使う（親近感が増す）
- [ ] チャンネル説明文に「誰のための・何を学べる・更新頻度」を明記
- [ ] 再生リストを柱ごとに3つ作成
- [ ] 最初の3本は公開前に撮りだめしておく

---

## 最初にやること（優先順）

1. 競合チャンネル5つをリサーチ → 再生数が多い動画のタイトル・サムネを分析
2. 最初の10本のタイトル案を書き出す（撮影前にタイトル先行で考える）
3. チャンネル開設 → アイコン・説明文を整備
4. 1本目を撮影・公開（完璧より「公開すること」が最優先）
